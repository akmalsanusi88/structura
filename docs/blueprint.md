# **App Name**: ConstructWise

## Core Features:

- Dashboard Summary: Dashboard overview displaying key project metrics and recent activity.
- Project Listing: Project listing and filtering by status (Setup, Planning, Implementation, Overdue, Closed).
- Project Setup: Project creation wizard allowing the user to manage project information and timeline.
- Plant Unit Management: Plant Units management interface: viewing the units as well as Create/Update/Delete (CRUD) capabilities.
- AI Cost Forecast: AI-powered cost forecasting: The application employs a tool that analyzes project parameters to estimate cost overruns or savings. This will use past data and project variables, with a high level of analysis.
- Report Generation: Simplified Report generation for visualizing key metrics.

## Style Guidelines:

- Primary color: Soft indigo (#667EEA) for a professional and trustworthy feel.
- Background color: Light gray (#F7FAFC) to provide a clean and spacious canvas.
- Accent color: Muted violet (#9F7AEA) for highlighting interactive elements and key information.
- Body font: 'PT Sans', a sans-serif, for a warm and modern aesthetic and great readability in the application body.
- Headline font: 'Space Grotesk', a sans-serif, for headlines to convey a computerized feel, complementing the precision in project management.
- Use clear, minimalist icons from Lucide React to represent various actions and modules.
- Implement a responsive grid system optimized for mobile, tablet, and desktop screens.
- Subtle transitions and animations using Tailwind CSS to enhance user interaction and provide feedback.
